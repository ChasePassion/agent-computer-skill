# Windows 文件资源管理器手册
> 适用：Windows 11 | 更新：2026-03-14

## 执行原则
1. 优先快捷键，减少无效点击
2. 已知路径时优先地址栏直达，不逐层展开
3. 搜索前先进入最可能的目标目录
4. 批量操作优先多选 + 标准快捷键
5. 仅在任务需要时切换视图

---

## 界面元素与行为

| 元素 | 操作 | 结果 |
| :--- | :--- | :--- |
| 左侧导航窗格 | 点击节点（下载、C: 等） | 主内容区跳转到目标位置 |
| 主内容区 | 双击文件夹 | 进入下一级目录 |
| 主内容区 | 单击文件/文件夹 | 选中，状态栏显示信息 |
| 地址栏面包屑 | 点击某个父级目录名 | 直接跳回该层级 |
| 地址栏下拉入口 | 点击 | 展开访问历史列表 |
| 地址栏空白处 | 点击 | 切换为可编辑纯文本路径 |
| 搜索框 | 输入关键词 | 在当前目录及子目录中筛选 |
| 右下角视图切换区 | 点击图标 | 切换文件展示方式 |

---

## 常用流程

**导航**：双击进入 → 地址栏面包屑跳回父级 / 左侧导航栏跨盘切换

**精准搜索**：先进入目标目录 → 点击搜索框 → 输入关键词 → 双击结果打开

**历史跳转**：点击地址栏下拉入口 → 从历史列表中点击目标路径

**切换视图**：右下角图标切换（详细信息 = 日期/类型/大小；大图标 = 缩略图预览）

## 快捷键速查

### 窗口与标签页
| 快捷键 | 功能 |
| :--- | :--- |
| `Win + E` | 打开新窗口 |
| `Ctrl + N` | 打开独立新窗口 |
| `Ctrl + T` | 新建标签页 |
| `Ctrl + W` | 关闭当前标签页（最后一个则关闭窗口） |
| `Ctrl + Tab` / `Ctrl + Shift + Tab` | 切换标签页 |

### 导航与路径
| 快捷键 | 功能 |
| :--- | :--- |
| `Alt + ↑` | 返回上一级 |
| `Alt + ←` / `Backspace` | 返回上一页 |
| `Alt + →` | 前进 |
| `Alt + D` | 定位到地址栏并选中路径 |
| `F4` | 展开地址栏历史列表 |
| `Tab` | 在导航栏/内容区/地址栏/搜索框间切换焦点 |
| `Enter` | 打开选中项 / 进入文件夹 |

### 搜索与视图
| 快捷键 | 功能 |
| :--- | :--- |
| `Ctrl + E` / `F3` | 定位到搜索框 |
| `Ctrl + 滚轮` | 缩放文件图标 |
| `Alt + P` | 显示/隐藏预览窗格 |
| `Alt + Shift + P` | 显示/隐藏详细信息窗格 |

### 文件操作
| 快捷键 | 功能 |
| :--- | :--- |
| `Ctrl + Shift + N` | 新建文件夹 |
| `F2` | 重命名 |
| `Ctrl + A` | 全选 |
| `Ctrl + C` / `X` / `V` | 复制 / 剪切 / 粘贴 |
| `Delete` | 删除到回收站 |
| `Shift + Delete` | 永久删除 |
| `Alt + Enter` | 打开属性 |

### 多选技巧
- `Shift + 单击`：连续选中区间内所有文件
- `Ctrl + 单击`：逐个选中不相邻文件
