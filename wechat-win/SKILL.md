# 微信电脑版（Windows）操作手册
> 适用：Windows 桌面端微信客户端

## 1. 核心原则

1. 拖拽发送文件是最高效的本地文件分享方式，优先于其他方法
2. 右键菜单是消息处理的主要入口，多数操作均从此发起
3. 群聊管理、聊天记录查找等功能均从右上角 "..." 展开的信息面板进入
4. 涉及支付校验、账单详情或更高权限安全校验时，转手机端操作
5. 发送键行为（Enter / Ctrl+Enter）可在设置中切换

---

## 2. 主要使用流程

以下流程覆盖该应用的大部分实际使用场景。

### 主流程 A：消息处理

1. **复制 / 转发 / 收藏 / 引用 / 翻译 / 另存为**：在任意聊天气泡右键 → 选择对应操作
2. **划词操作**：选中气泡中部分文字 → 浮现快捷工具条 → 复制 / 翻译 / 搜一搜
3. **多选转发**：右键 → 多选 → 勾选多条记录 → 点击下方"转发" → 选择逐条或合并转发
4. **图片 / 文件操作**：右键图片、视频、文件消息 → 另存为 / 转发 / 收藏
5. **发送表情**：输入框点击笑脸图标 → 支持系统默认表情及自定义收藏表情

### 主流程 B：文件传输

1. **拖拽发送（优先）**：将本地文件或文件夹直接拖入聊天输入框发送
2. **文件传输助手**：在会话列表找到"文件传输助手"专用会话，用于手机与电脑之间快速同步文件、代码文本或链接

### 主流程 C：群聊管理

入口：点击对话窗口右上角 **"..."** → 展开右侧聊天信息面板。

1. **查看群信息**：面板中可查看群成员、群公告、聊天文件、聊天记录
2. **@ 成员**：在输入框输入 @ 后选择成员
3. **添加成员发起群聊**：面板中点击 **"+"** 添加更多联系人
4. **联系人管理**：点击好友头像查看资料；点击朋友圈相册图标浏览历史动态；可设置备注和标签
5. **置顶聊天 / 消息免打扰**：面板中对应开关控制

### 主流程 D：聊天记录查找

入口：聊天信息面板 → 查找聊天记录气泡 → 打开独立检索窗口。

1. **分类检索**：按文件 · 图片与视频 · 链接 · 音乐 · 小程序 · 视频号分类浏览
2. **按日期定位**：使用日期面板快速跳转到指定年月日
3. **清空聊天记录**：聊天记录管理中选择"清空聊天记录"，用于保护隐私或释放存储空间

### 主流程 E：收藏与小程序

1. **收藏夹**：侧边栏"收藏"面板 → 按标签或类型（链接 / 相册 / 语音 / 笔记）筛选 → 可直接转发给好友或删除
2. **小程序**：可在侧边栏或收藏夹中打开，小程序在独立窗口运行

## 3. 稳定规则

### 3.1 路由 / 页面 / 模块

- **聊天信息面板**：入口为对话窗口右上角 "..."，包含群管理、聊天记录查找、置顶、免打扰等所有会话级设置
- **收藏夹**：位于左侧侧边栏，独立面板
- **文件传输助手**：会话列表中的专用系统会话
- **公众号**：独立面板，支持瀑布流阅读最新推送
- **服务通知 / 微信支付**：左侧栏实时同步交易凭证

### 3.2 参数 / 状态 / 业务语义

- 多选转发支持"逐条转发"和"合并转发"两种方式
- 收藏内容支持按类型筛选：链接 / 相册 / 语音 / 笔记
- 小程序运行在独立窗口，不在主聊天窗口内
- 支付校验、账单详情、更高权限安全校验：需转至手机端完成
- 公众号部分功能需更高权限时，按提示转手机端操作

### 3.3 判断标准

- **直接通过（桌面端完成）**：普通消息收发、文件传输、聊天记录查找、收藏管理、群聊操作
- **条件通过（优先桌面，不行转手机）**：公众号高权限操作
- **直接跳转手机端**：支付校验、账单详情、安全确认类操作

## 4. UI / 坐标 / 页面映射

### 4.1 页面区域

- **左侧侧边栏**：会话列表、收藏、公众号、服务通知 / 微信支付等入口
- **中央聊天窗口**：消息气泡列表 + 底部输入框（含笑脸表情按钮）
- **右上角 "..." 按钮**：展开聊天信息面板（群管理 / 联系人详情 / 聊天记录等）
- **右侧聊天信息面板**：群成员、群公告、查找聊天记录、置顶 / 免打扰开关、"+" 添加成员
- **独立检索窗口**：从聊天信息面板进入，用于分类 / 按日期检索聊天记录

### 4.2 关键控件 / 坐标

| 元素 | 坐标 / 位置 | 说明 |
| :--- | :--- | :--- |
| "..." 按钮 | 对话窗口右上角 | 展开右侧聊天信息面板 |
| "+" 添加成员 | 聊天信息面板内 | 添加联系人发起或扩展群聊 |
| 笑脸图标 | 输入框左侧工具栏 | 打开表情面板 |
| 收藏入口 | 左侧侧边栏 | 打开收藏夹面板 |
| 查找聊天记录气泡 | 聊天信息面板内 | 打开独立检索窗口 |

### 4.3 反馈映射

| 动作 | 预期反馈 | 含义 | 下一步 |
| :--- | :--- | :--- | :--- |

## 5. 快捷键

| 快捷键 | 功能 | 备注 |
| :--- | :--- | :--- |
| `Ctrl + Alt + W` | 打开 / 隐藏微信主界面 | 全局；可在设置 → 快捷键中修改 |
| `Alt + A` | 截图 | 全局；支持识别文字、打码、框选；可修改 |
| `Ctrl + L` | 锁定微信 | 全局；解锁需手机端确认 |
| `Ctrl + F` | 全局搜索 | 聚焦搜索框，查找联系人 / 群聊 / 聊天记录 |
| `Ctrl + Win`（长按） | 全局语音输入 | 语音转文字；跨软件全局可用 |
| `Enter` | 发送消息 | 可在设置中切换为 `Ctrl + Enter` 发送 |
| `Shift + Enter` | 换行 | 发送键设置为 `Enter` 时使用 |
| `Ctrl + 鼠标滚轮` | 缩放文字 / 图片 | 放大或缩小聊天窗口内字号及图片尺寸 |
| `Esc` / `Alt + F4` | 关闭当前窗口 | 关闭独立聊天窗口或图片 / 视频预览窗口 |

## 6. 常见失败与排障

### 现象：文件发送失败 / 无法拖入输入框

- 可能原因：
  - 文件路径含特殊字符或文件正被占用
  - 拖拽目标区域不正确（未落入输入框）
- 优先检查：
  1. 文件是否正在被其他程序占用
  2. 拖拽落点是否准确落入聊天输入框区域
- 处理方式：
  - 确认文件未被占用后重试；若拖拽失败，改用输入框工具栏中的文件发送按钮

### 现象：支付 / 安全校验无法在桌面端完成

- 可能原因：
  - 桌面端不支持支付校验、账单详情等高安全级别操作
- 优先检查：
  1. 操作类型是否属于支付校验 / 安全确认 / 账单详情
- 处理方式：
  - 直接转至手机端完成对应操作

### 现象：锁定微信后无法在桌面端解锁

- 可能原因：
  - `Ctrl + L` 锁定后，解锁需手机端扫码或确认
- 优先检查：
  1. 手机微信是否在线且可操作
- 处理方式：
  - 打开手机微信，按提示确认解锁
