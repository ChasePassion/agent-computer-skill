# 微信电脑版（Windows）操作手册
> 适用：Windows 桌面端微信客户端

## 消息处理

- **右键菜单**：在任意聊天气泡右键 → 复制 / 转发 / 收藏 / 引用 / 翻译 / 另存为（图片/文件）
- **划词操作**：选中气泡中部分文字 → 浮现快捷工具条 → 复制 / 翻译 / 搜一搜
- **多选转发**：右键 → 多选 → 勾选多条记录 → 点击下方"转发" → 选择逐条或合并转发
- **图片/文件**：右键图片、视频、文件消息 → 另存为 / 转发 / 收藏
- **发送表情**：输入框点击笑脸图标 → 支持系统默认表情及自定义收藏表情

## 文件传输

- **拖拽发送**：直接将本地文件或文件夹拖入聊天输入框发送，是最高效的分享方式
- **文件传输助手**：手机与电脑之间快速同步文件、代码文本或链接的专用会话


## 群聊管理

入口：点击对话窗口右上角 **"..."** 展开右侧聊天信息面板。

- **查看信息**：群成员、群公告、聊天文件、聊天记录
- **群聊操作**：查看群成员、阅读群公告、@ 成员
- **联系人管理**：备注、标签；点击好友头像可查看资料，点击朋友圈相册图标可浏览历史动态
- **发起群聊**：面板中点击 **"+"** 添加更多联系人
- **置顶聊天 / 消息免打扰**：面板中开关控制

## 聊天记录查找

入口：聊天信息面板 → 查找聊天记录气泡 → 独立检索窗口。

- **分类检索**：文件 · 图片与视频 · 链接 · 音乐 · 小程序 · 视频号
- **按日期定位**：日期面板快速跳转到指定年月日
- **清空聊天记录**：聊天记录管理中可选择"清空聊天记录"，保护隐私或释放存储空间

## 收藏与小程序

- **收藏夹**：侧边栏"收藏"面板支持按标签、类型（链接 / 相册 / 语音 / 笔记）筛选；选中内容可直接转发给好友或删除
- **小程序**：可在侧边栏或收藏夹中打开，小程序在独立窗口运行

## 其他功能

- **公众号**：独立面板支持瀑布流阅读最新推送；需更高权限或安全校验时按提示转手机端完成
- **服务通知 / 微信支付**：左侧栏实时同步交易凭证；涉及支付校验或账单详情时使用手机端操作

## 快捷键

| 快捷键 | 功能 | 备注 |
| :--- | :--- | :--- |
| `Ctrl + Alt + W` | 打开 / 隐藏微信主界面 | 全局，可在设置 → 快捷键中修改 |
| `Alt + A` | 截图 | 全局，支持识别文字、打码、框选，可修改 |
| `Ctrl + L` | 锁定微信 | 全局，解锁需手机端确认 |
| `Ctrl + F` | 全局搜索 | 聚焦搜索框，查找联系人 / 群聊 / 聊天记录 |
| `Ctrl + Win`（长按） | 全局语音输入 | 语音转文字，跨软件全局可用 |
| `Enter` | 发送消息 | 可在设置中切换为 `Ctrl + Enter` 发送 |
| `Shift + Enter` | 换行 | 发送键设置为 `Enter` 时使用 |
| `Ctrl + 鼠标滚轮` | 缩放文字/图片 | 放大或缩小聊天窗口内字号及图片尺寸 |
| `Esc` / `Alt + F4` | 关闭当前窗口 | 关闭独立聊天窗口或图片/视频预览窗口 |
